The manifesto

Built like
a product,
not a funnel.

No "free tier vs pro tier". No popups. No "wait 5s for download". This is what tool sites should have felt like in 2014, and what they should feel like now.

01

Everything runs in your browser.

Your files never touch a server. Close the tab, the data is gone. Most "online" converters can't promise that. We can — open the network tab and look.

02

One ad per page, max.

Yes, this site has ads — that's how it stays free. But you'll see one, clearly labeled, never autoplay video, never interstitials. The tool is the product.

03

Designed once, shipped daily.

Every tool uses the same design system, so once you learn one, you know them all. New tool roughly every 5 days. Suggestions get built fast.

04

Open source where it counts.

The core engine, the tool framework, the design tokens — all MIT licensed on GitHub. Fork it, copy it, learn from it. Send a PR for your favorite tool.
